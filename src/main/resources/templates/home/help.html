Ayuda

Flujo recomendado
  1. Crear un proyecto de estimación.
  2. Definir una técnica de tamaño: Puntos Función o UCP.
  3. Completar los datos del análisis.
  4. Crear una conversión a esfuerzo mediante Delphi o función de transformación.
  5. Revisar el esfuerzo y, si existe tarifa por hora, el coste estimado.
Proyectos

Cada estimación parte de un proyecto. En él se puede indicar nombre, descripción y una tarifa por hora opcional para calcular costes.

Técnicas de tamaño
Puntos Función

Permite estimar el tamaño funcional a partir de funciones de datos, funciones transaccionales y características generales del sistema.

Use Case Points

Permite estimar el tamaño a partir de actores, casos de uso, factores técnicos y factores ambientales.

Conversión a esfuerzo
Delphi

Utiliza estimaciones de expertos sobre módulos de menor y mayor tamaño para obtener una función lineal de esfuerzo.

Funciones de transformación

Permiten aplicar una fórmula lineal del tipo E = a + b × tamaño para convertir tamaño en horas-persona.

Coste

Si el proyecto tiene configurada una tarifa por hora, el sistema calcula automáticamente el coste estimado multiplicando las horas-persona por dicha tarifa.
