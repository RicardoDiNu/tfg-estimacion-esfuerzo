Sistema para estimación en proyectos de software

Aplicación web para estimar tamaño, esfuerzo y coste de proyectos software mediante técnicas como Puntos Función, UCP, Delphi y funciones de transformación.

Gestión de proyectos

Crea proyectos de estimación y organiza la información necesaria para aplicar distintas técnicas.

Estimación de tamaño

Calcula el tamaño funcional mediante Puntos Función y Use Case Points.

Conversión a esfuerzo

Convierte el tamaño estimado en horas-persona mediante Delphi o funciones de transformación.

Estimación de coste

Obtén una estimación económica a partir del esfuerzo calculado y la tarifa por hora del proyecto.

Flujo general de uso
1. Proyecto

Crea un proyecto de estimación.

2. Tamaño

Aplica PF o UCP.

3. Esfuerzo

Usa Delphi o una función.

4. Coste

Calcula el coste estimado.
