Sobre el proyecto

Esta aplicación permite estimar tamaño, esfuerzo y coste de proyectos software.

El sistema está desarrollado como Trabajo Fin de Grado.

Técnicas incluidas
  • Estimación de tamaño mediante Puntos Función.
  • Estimación de tamaño mediante Use Case Points.
  • Conversión de tamaño a esfuerzo mediante Delphi.
  • Conversión de tamaño a esfuerzo mediante funciones de transformación.
  • Estimación de coste a partir de horas-persona y tarifa por hora.
