Sobre la aplicación

Esta aplicación web permite gestionar proyectos de estimación de software y calcular su tamaño, esfuerzo y coste.

El sistema reúne en una misma herramienta la gestión de proyectos, la aplicación de técnicas de estimación, la conversión a esfuerzo y la generación de resultados.

La aplicación está desarrollada como Trabajo Fin de Grado en Ingeniería Informática del Software.

Alcance funcional
  • Gestión de proyectos de estimación.
  • Gestión de usuarios, roles, trabajadores y permisos por proyecto.
  • Estimación de tamaño mediante Puntos Función y Use Case Points.
  • Conversión de tamaño a esfuerzo mediante Delphi y funciones de transformación.
  • Estimación de coste a partir de horas-persona y tarifa horaria.
  • Generación de informes PDF e importación/exportación XML.
Técnicas incluidas
Técnicas de tamaño
  • Puntos Función.
  • Use Case Points.
Técnicas de conversión a esfuerzo
  • Delphi.
  • Funciones de transformación.
Base técnica

La aplicación está desarrollada como una aplicación web Java con Spring Boot, Thymeleaf, Spring Security, JPA/Hibernate y PostgreSQL.

También puede ejecutarse mediante Docker para facilitar su puesta en marcha en otros equipos.
